Let we Create simple  Adaptive Layout app… For Portrait and Landscape mode



Control+alt+command+3 to show Object Library search UIView drag and drop two views into controller Name it YellowView and BlueView




Select YellowView and Choose Pin from Auto layout bar Uncheck Constrain to margin and click the Top, Leading and Trailing Constraints  and change those values into 0(zero), Click Add 3 constraints.


Select BlueView and Choose Pin from Autolayout bar Uncheck Constrain to margin

and click the Bottom, Leading and Trailing Constraints  and change those values into 0(zero), Click Add 3 constraints.






Now in Document Outline it show Constraints missing and Misplaced Views. Don’t worry we will clear it now….


Right click from BlueView and drag move to yellow view, choose vertical spacing.
and again Right click from BlueView and drag move to yellow view, choose equal height.

 Or 

 Right click from BlueView and drag move to yellow view, Press SHIFT and click both equal height and vertical spacing.


Click Resolve Auto layout issues from Autolayout bar. 

Select any view from controller otherwise the menus are disabled.
          Click  All views in view controller —>  Updates frames.



There is some distance between Yellow and Blue view. Select BlueView Or YellowView change the Top Space to YellowView or Bottom Space to BlueView —> 0  from Size Inspector. (Don’t confuse both constraints are same)

Run the app... We Completed half.

Change orientation to Landscape wow Blank screen… Let we Design Layout for Landscape....


Change the Size class to w Any | h Compact

Here also Blank screen, select BlueView in Document Outline, goto attribute inspector at last you can see… installed option, click + button Now select  Any Width | Compact Height (current) and 
tick the checkbox (w Any h C) now BlueView show in screen.





Do the same for YellowView... Now YellowView show in Viewcontroller Palce the YellowView in left and BlueView in Right.


Select YellowView

we have to Add Top, Leading and Bottom Constraint to it. 

In Document Outline expand Constraints, There are many constraints right, Don't confuse, in Document outline below there is a search bar let search for YellowView.

Now Select Vertical spacing - YellowView - Top Layout Guide, Goto Attribute Inspector Click + button and Select Any Width | Compact Height (current) and tick the checkbox (w Any h C). This Top (vertical )Constraint is installed for Both landscape and portrait.

Then Select Leading Alignment - YellowView - View, Goto Attribute Inspector Click + button and Select Any Width | Compact Height (current) and tick the checkbox (w Any h C). This Leading Constraint is installed for Both landscape and portrait.

Select YellowView and Choose Pin from Autolayout bar Uncheck Constrain to margin and click the Bottom Constraints  and change those values into 0(zero), Click Add 1 constraints.


Search blueview in document Outline 

Select BlueView

Now Select Vertical spacing -Bottom Layout Guide - BlueView Goto Attribute Inspector Click + button and Select Any Width | Compact Height (current) and tick the checkbox (w Any h C). This Bottom (vertical )Constraint is installed for Both landscape and portrait.

Then Select Trailing Alignment - View - YellowView, Goto Attribute Inspector Click + button and Select Any Width | Compact Height (current) and tick the checkbox (w Any h C). This Trailing Constraint is installed for Both landscape and portrait.


Right Click and drag from YellowView to BlueView, Press SHIFT and Select Horizontal Spacing, Top and Equal widths.


Click Resolve Auto layout issues from Autolayout bar. 

Select any view from controller otherwise the menus are disabled.
          Click  All views in view controller —>  Updates frames.


Now Run.... Oh yeah its working....
